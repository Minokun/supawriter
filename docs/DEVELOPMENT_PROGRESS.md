## 项目概览

**SupaWriter** - AI智能写作助手，支持文章生成、风格学习、SEO优化、多平台发布。

### 技术栈
- **后端**: FastAPI + PostgreSQL + Redis + ARQ
- **前端**: Next.js 14 + React + Tailwind CSS
- **AI**: DeepSeek / OpenAI / 自定义LLM
