## 一、产品背景

P0 完成后，SupaWriter 已具备爆款基因（热点→一键写作、多平台适配、质量评分、风格学习、增长飞轮）。P1 的目标是**构建付费用户愿意买单的不可替代价值**，实现商业化闭环。

**P1 核心命题**: 让用户觉得"不付费就亏了"

**来自 Research Report 的关键发现**:
- SEO优化和批量生产是付费用户最愿意买单的功能（§3.2 竞品对比）
- 竞品定价 ¥99-299/月，SupaWriter Pro ¥49/月有价格优势但需功能支撑（§6.2）
- 写作Agent是 2025-2026 最大趋势，率先实现可建立技术壁垒（§3.4）

### F7: SEO优化引擎

#### 3.1.1 功能概述

为文章提供全方位SEO优化能力，包括关键词密度分析、标题优化建议、元描述自动生成、内链建议。这是竞品（如此AI、5118）的核心卖点，也是付费用户最愿意买单的功能。

#### 3.1.2 用户故事

> 作为一个内容营销人员，我生成文章后点击"SEO优化"，系统分析关键词密度为1.2%（建议2-3%），给出3个标题优化建议，自动生成元描述，并建议在第2段插入内链。我按建议修改后，文章的搜索排名明显提升。

#### 3.1.3 功能模块

**A. 关键词分析**

| 功能 | 规格 |
|------|------|
| 关键词提取 | 自动从文章中提取3-5个核心关键词 |
| 密度分析 | 计算每个关键词的出现频率和密度百分比 |
| 密度建议 | 理想密度2-3%，过低/过高均给出调整建议 |
| 分布热力图 | 可视化关键词在文章各段落的分布情况 |
| 长尾词建议 | 基于核心关键词推荐3-5个长尾关键词 |

**B. 标题优化**

| 功能 | 规格 |
|------|------|
| 标题评分 | 从SEO角度评分（0-100）：关键词位置、长度、吸引力 |
| 优化建议 | 生成3个优化后的标题供用户选择 |
| 字数检查 | 建议标题长度15-30字（搜索引擎最佳展示长度） |
| 关键词前置 | 检查核心关键词是否在标题前半部分 |

**C. 元描述生成**

| 功能 | 规格 |
|------|------|
| 自动生成 | 基于文章内容自动生成120-160字的元描述 |
| 关键词包含 | 确保元描述包含核心关键词 |
| 可编辑 | 用户可修改自动生成的元描述 |
| 复制按钮 | 一键复制元描述 |

**D. 内链建议**

| 功能 | 规格 |
|------|------|
| 锚文本建议 | 识别文章中适合添加内链的关键短语 |
| 位置标注 | 在文章中标注建议添加内链的位置 |
| 相关文章推荐 | 从用户历史文章中推荐可互相链接的文章 |

#### 3.2.1 功能概述

### F9: 写作Agent（自动模式）

#### 3.3.1 功能概述

用户设定主题领域和生成频率，Agent自动追踪热点、选题、生成文章。实现"设定一次，持续产出"的全自动写作。

#### 3.3.2 用户故事

> 作为一个科技领域博主，我创建一个Agent：关注"AI/科技"领域，每天自动生成1篇文章。Agent每天早上自动检查热点，选择最相关的话题，生成一篇文章放到我的草稿箱。我只需每天审核发布即可。

#### 3.3.3 功能规格

**Agent配置**:

| 参数 | 规格 |
|------|------|
| 关注领域 | 用户自定义关键词/领域（如"AI、科技、创业"） |
| 生成频率 | 每天1篇 / 每周3篇 / 每周1篇 / 自定义cron |
| 写作风格 | 复用用户风格库（P0 F5） |
| 目标平台 | 公众号/知乎/小红书（决定格式和字数） |
| 信息源 | 热点追踪（默认） / 自定义RSS（P1 F12扩展） |
| 审核模式 | 自动生成到草稿箱（需人工审核发布） |

**Agent运行流程**:

```
定时触发（cron）
    ↓
检查热点/信息源 → 筛选与关注领域相关的话题
    ↓
AI选题评估 → 选择最佳话题
    ↓
调用文章生成pipeline（复用现有能力）
    ↓
生成完成 → 自动评分 → 存入草稿箱
    ↓
推送通知用户："您的Agent生成了新文章"
```

### F10: 热点预警推送

#### 3.4.1 功能概述

用户设定关注领域，当相关热点出现时实时推送通知，帮助用户第一时间蹭热点。

| 参数 | 规格 |
|------|------|
| 关注关键词 | 最多10个关键词/短语 |

| 推送渠道 | 站内通知（P1）→ 微信/邮件/Webhook（P2扩展） |

**匹配逻辑**: 热点标题/摘要包含用户关注关键词（模糊匹配+同义词扩展）

### F11: 数据看板

#### 3.5.1 功能概述

为用户提供创作数据统计和分析，包括文章生成统计、质量趋势、热点命中率等，帮助用户量化创作成果并优化策略。

#### 3.5.2 用户故事

> 作为一个日更博主，我打开数据看板看到本月生成了28篇文章，平均质量评分从72提升到81，热点命中率35%。我发现"科技"领域的文章评分最高，决定加大这个方向的产出。

#### 3.6.2 定价方案（对齐 Research Report §6.2）

| 等级 | 价格 | 文章配额 | 核心权益 |

| **Pro** | ¥49/月（年付¥39/月） | 30次/月 | 中级模型+SEO优化+多平台+热点预警+去水印+数据看板 |
| **Ultra** | ¥99/月（年付¥79/月） | 100次/月 | 顶级模型+批量生成+写作Agent+知识库(P2)+API(P2)+优先支持 |

| 额度包 | 价格 | 适用场景 |

| 50次额度包 | ¥79.9 | 短期大量需求 |

#### 3.6.3 功能规格

**A. 定价页面**

| 元素 | 规格 |
|------|------|
| 布局 | 三列对比卡片（Free/Pro/Ultra） |
| 高亮 | Pro卡片标记"最受欢迎" |

| FAQ | 底部常见问题折叠面板 |

**B. 支付流程**

| 步骤 | 规格 |
|------|------|
| 选择方案 | 点击"立即升级"→确认方案和价格 |
| 支付方式 | 微信支付 + 支付宝（优先接入微信） |
| 支付结果 | 成功→即时生效+欢迎页；失败→重试引导 |
| 发票 | 支持开具电子发票（可后续迭代） |

**C. 订阅管理**

| 功能 | 规格 |
|------|------|
| 当前方案 | 设置页显示当前等级、到期时间、剩余配额 |

| 续费 | 到期前7天提醒，支持自动续费 |
| 取消 | 可随时取消，当前周期内权益保留 |
| 额度包购买 | 独立于订阅，购买后立即生效 |

**D. 配额控制**

| 场景 | 处理方式 |
|------|---------|
| 配额用完 | 提示"本月配额已用完"，引导升级或购买额度包 |
| 即将用完 | 剩余≤3次时在写作页显示提醒 |
| 额度包叠加 | 额度包额度独立于月配额，优先消耗月配额 |
