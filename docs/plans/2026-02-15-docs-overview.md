# SupaWriter 迭代文档说明总览（P0-P3）

> **创建日期**: 2026-02-15  
> **用途**: 开发团队快速上手 + 迭代进度记录  
> **覆盖范围**: `docs/plans/` 本次全部方案文档（PRD / 技术设计 / Sprint / 总纲）+ 关键系统架构文档

---

## 1. 使用说明（先看）

本总览文档解决两个问题：

1. **快速了解**：新成员 10-20 分钟内知道“先看什么、每份文档讲什么”。
2. **进度记录**：按阶段（P0-P3）持续更新“文档状态 / 研发状态 / 风险与阻塞”。

建议作为 **唯一入口索引** 使用，后续每周例会更新一次。
