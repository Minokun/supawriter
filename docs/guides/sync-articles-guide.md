# 一键发布文章到数据库使用指南

## 功能位置
历史记录页面 → 📤 一键发布到数据库

## 使用流程

### 1. 检查同步状态
点击"🔍 检查同步状态"按钮，查看本地文章与数据库的同步情况。

### 2. 选择文章
- 单选：勾选要发布的文章
- 全选：使用"🎯 全选"功能

### 3. 发布
点击"🚀 发布选中文章"按钮，系统将批量上传到PostgreSQL数据库。

## 同步规则
- 通过"标题+日期"判断是否已同步
- 保留原始创建时间
- 本地文章不会被删除

## 注意事项
- 需要配置PostgreSQL数据库才能使用此功能
- 同步失败的文章会显示详细错误信息
- 可重复执行同步操作
